Business Rules — End-to-End Demo

Complete walkthrough of Fabric Ontology Business Rules CRUD flows.
PRD: WS-1915347 — Fabric Ontology Business Rules · Domain: ZavaStadium Ontology

User Journey Map
Flow 1 — Create a new rule
📋 Rules List → [New rule] → ✏️ Create Rule → [Save] → ✅ List + Toast
Validation branch on Save:
[No entity type] → ⚠️ Validation Gate → [Tag & Save] → ✅ List + Toast
Flow 2 — Edit an existing rule
📋 Rules List → [Click rule] → ✏️ Edit Rule → [Save] → ✅ List + Toast
Flow 3 — Delete a rule
✏️ Edit Rule → [Delete] → 🗑 Delete Confirm → [Confirm] → ✅ List + Toast
Flow 4 — Discover rules from entity type
🔷 Entity Type Detail → [Click rule in Business Rules section] → ✏️ Edit Rule
