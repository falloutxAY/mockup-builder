Implementation notes

Pattern Details
Concept browser (two-panel picker)Standard Fluent components in a flex layout: Dialog + SearchBox + Checkbox + Divider + Button. No custom widgets needed.
NL editor with intellisenseThe text editor is custom (contenteditable div) — devs build: concept detection logic, inline highlighting, and cursor-positioned dropdown. The dropdown itself uses Fluent Combobox (freeform) + OptionGroup. Hover tooltips use Fluent Tooltip.

Icon implementation

Mockups use fluent-icons.css — a CSS mask-image system where each icon is an inline SVG data URI that inherits currentColor. In production, replace with @fluentui/react-icons using bundleIcon(FilledIcon, RegularIcon) for filled/regular pairs. Class pattern: <span class="fi fi-sm fi-{name}"></span> — sizes: fi-xs (12px), fi-sm (16px), default (20px), fi-lg (24px).
